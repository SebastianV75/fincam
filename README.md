# Fincam

Fincam is a simple personal finance web app built mobile-first for iPhone and also usable on laptop.

The product is focused on helping users:

- organize each paycheck period,
- track daily spending quickly,
- separate money into fixed bills, credit card, savings, and free spending,
- manage accounts and savings goals with a calm, clear interface.

## Current Project Status

This repository currently contains the initial product planning and design foundation:

- product brief,
- MVP definition,
- screen wireframes,
- detailed mockups,
- mini design system.

## Planned Stack

- Next.js
- TypeScript
- Tailwind CSS
